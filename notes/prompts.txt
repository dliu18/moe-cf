The purpose of this script is to evaluate the impact of data mixing the in-context prompt on group preference alignment. The script takes the following as input:

* test_prompts: the number of test prompts to include. Defaults to -1, the maximum.

* num_trials: number of trials to perform for each value of alpha. Default to 1

We leverage the Community Alignment dataset which contains pregenerated prompts as well as the preferred responses (among 4 options) of human annotators. We tune the LLM by providing example prompts and preferred responses before evaluating on unseen prompts. To evaluate, we query the LLM with unseen prompts and see if the chosen response matches those selected by annotators from the source group. The key is to determine whether data mixing (the ratio of example prompts from the source group, as opposed to the augmentation group) impacts performance. More details here: 

Data mixing is controlled by a parameter alpha. alpha denotes the fraction of example prompts from the source group. The script should scan over possible values of alpha.
